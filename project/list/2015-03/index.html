二、服务内容：
2.1服务条款
2.1.1本条款所称“服务”特指用户通过寰宇医道提交预约信息后预约特定医生进行的健康咨询及相关活动，该服务的交互主体为寰宇医道的用户和寰宇医道。
2.1.2寰宇医道需保证提供安全、稳定的服务场所，保证服务的顺利进行。
2.1.3如服务在进行过程中由于寰宇医道平台性能不稳定等系统原因导致服务不能完成的，将由寰宇医道为您安排重新健康咨询服务。
2.1.4一旦发现用户提供的个人信息中有虚假，寰宇医道有权立即终止向用户提供的所有服务，并冻结用户的帐户，有权要求用户赔偿因提供虚假信息给寰宇医道造成的损失。
2.1.5健康咨询均只限于根据用户提供的病历资料，医生尽可能利用医学知识及临床经验给予一定的解惑及如何就医方面的建议。寰宇医道不保证满足用户要求的诊断、治疗方面的建议一定能获得，不对结果是否符合用户预期做保证。
2.1.6健康咨询中医生所提供内容均是个人建议，不得做为诊断、治疗的直接医疗处置，用户需知晓并同意诊断及治疗均需前往医院。擅自将医生建议做为处方使用的，后果自负，与医生及寰宇医道无关。
2.1.7用户在健康咨询过程中要语言文明，尊重医生，平等交流。如有对医生恶意中伤，语言不文明，医生有权立即中断服务，用户无权要求退款。
2.1.8健康咨询过程中遇到医生有紧急事情处理时（医生职业要求），要给予理解，暂停咨询，向寰宇医道客服申报，以便安排重新咨询的具体时间。

2.2.1寰宇医道有义务在现有技术上维护服务的正常进行，并努力提升技术及改进技术，使服务更好进行。
2.2.2寰宇医道必须保证提供健康咨询服务的医生是平台介绍的医生本人。
2.2.3对于用户在本平台预定服务中的不当行为或其它任何寰宇医道认为应当终止服务的情况，寰宇医道有权随时作出终止服务等处理，而无须征得用户的同意。
如存在下列情况：
(1) 用户或其它第三方通知寰宇医道，认为某个具体用户或具体交易事项可能存在重大问题；
(2)用户或其它第三方向寰宇医道告知咨询内容有违法或不当行为的，寰宇医道以普通非专业的知识水平标准对相关内容进行判别，可以明显认为这些内容或行为具有违法或不当性质的。

2.2.5寰宇医道有权根据不同情况选择是否停止对该用户提供服务，并追究相关法律责任。
2.2.6当用户顺利使用完付费咨询服务后不得因为咨询服务中的内容不满意要求退款。
2.2.7系统因下列状况无法正常运作，使用户无法使用咨询服务时，寰宇医道不承担损害赔偿责任，该状况包括但不限于：

(4) 因用户自身原因错过咨询时间或其他不可控原因导致服务不能使用的。
三、隐私声明：
3.1 适用范围
3.1.1 您预约申请寰宇医道服务时，根据要求提供的个人信息；
3.1.2 您使用寰宇医道咨询服务、参加相关活动、访问网站网页（或移动客户端）时，网站自动接收并记录用户浏览器上的服务器数据，包括但不限于IP地址、网站Cookie中的资料及您要求取用的网页记录；
3.1.3 您接受服务时，提供的非公开的病历资料、基本情况、诊疗方案等文字、图片、数据信息。
3.1.4 寰宇医道通过合法途径从其他途径取得的您的个人资料。
3.1.5 您认为不宜公开的其他内容。
3.2 信息保密
3.2.1 保护您的隐私是寰宇医道的一项基本政策，寰宇医道保证不对外公开或向第三方提供您在使用网络服务时存储的非公开内容。
3.2.2 寰宇医道可能会与第三方合作向用户提供相关的网络服务，在此情况下，如该第三方同意承担与寰宇医道同等的保护用户隐私的责任，则寰宇医道有权将您的预约申请资料等提供给该第三方。
3.3 信息使用：
3.3.1 在不透露您的保密信息的前提下，寰宇医道有权对整个用户数据库进行分析并对用户数据库进行商业上的利用。
3.3.2 医患之间的咨询问题内容在隐去姓名、单位、地址等信息后，寰宇医道可以不经过您的授权，进行公开、编辑、出版、发行。
3.3.3 为服务用户的目的，寰宇医道可能通过使用您的个人信息向您提供服务，包括但不限于向您发出活动和服务信息等。
